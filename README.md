leemo.xyz is a hands-on project to learn web development and explore real-world application structure. The site uses Cloudflare Workers to securely handle API requests and connect to a Supabase database for data storage. It includes a simple to-do list system along with several API-driven features and UI components, including integrations such as the Duolingo API and Lanyard API. I also used it as a way to learn and experiment with Anthropic Claude for building and improving features through AI-assisted development. The project is intentionally experimental and not fully polished, focusing on learning over production quality while building experience with HTML, CSS, JavaScript, and modern web integration workflows.
